# 给大模型的阅读说明（益语软件平台源码）

本目录是“当前工作区快照”的源码导出，按分卷 Markdown 保存，适合大模型分批阅读。

## 目录结构

- `source-part-xxx.md`：源码分卷，共 **12** 卷。
- `INDEX.md`：路径索引（先查这个，再定位分卷）。
- `README.md`：阅读方法（本文件）。

## 大模型建议读取顺序

1. 根配置：`README.md`、`package.json`、`tsconfig*.json`、`vite.config.ts`。
2. 桌面前端：`src/`。
3. 本地后端：`backend/`。
4. 云端后端：`cloud_backend/`。
5. 工具脚本与部署：`scripts/`、`deploy/`。
6. 移动端：`mobile/`（按 `app/`、`components/`、`lib/`、`android/`、`ios/` 阅读）。

## 如何精准定位文件

1. 在 `INDEX.md` 搜索目标相对路径。
2. 找到对应分卷（如 `source-part-004.md`）。
3. 在该分卷搜索标题 `## ` + 文件路径。

## 上下文节省策略

- 先加载 `INDEX.md`，不要一次性加载全部分卷。
- 单任务仅加载相关模块；跨模块再补充依赖文件。
- 做代码评审时，优先加载“入口文件 + 调用链文件 + 对应测试”。
