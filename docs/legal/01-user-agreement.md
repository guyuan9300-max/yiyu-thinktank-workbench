## 第二章 服务说明

### 第 3 条 软件功能

3.1 本软件是一款面向个人或组织的本地优先（local-first）智能工作台软件，主要功能包括但不限于：

- **任务与日程管理**：创建、组织、跟踪个人或团队任务
- **战略陪伴**：基于您导入的资料生成组织级智能资产沉淀
- **资讯情报站**：聚合并智能筛选您订阅的资讯
- **成长中心**：知识卡片化沉淀与回顾
- **数据中心**：本地大模型优化与文档智能化处理
- **组织协作**：组织内成员之间的任务协作与信息流转（仅在云端模式下）
- **系统设置**：AI 引擎接入、组织/部门/角色管理、个人偏好

3.2 **本软件采用"本机优先 + 可选云端"双模式架构**：

- **本机模式（默认）**：您的所有数据存储在您本机操作系统的应用支持目录下（macOS: `~/Library/Application Support/YiyuThinkTankWorkbench2/`），**我们的服务器不存储您的任何业务数据**。本机模式可在没有任何网络连接的情况下使用大部分功能。
- **云端模式（可选）**：您可主动选择登录益语云账号，启用云同步、组织协作等功能。届时您在本软件中产生的业务数据将根据您的操作同步至我们的云端服务器。详细数据流见《[隐私政策](./02-privacy-policy.md)》。

3.3 您可随时在系统设置中查看当前模式，并在两种模式之间切换。从云端模式切换到本机模式时，已同步至云端的数据可由您选择删除或保留。

### 第 4 条 第三方服务依赖

4.1 本软件部分功能依赖第三方服务，包括但不限于：

- **AI 大模型服务**：当您启用智能化分析功能时，您输入的文本/文档片段将被发送至您在系统设置中配置的 AI 引擎处理。**当前默认接入的 AI 引擎为"豆豆 Seed 2.0 Pro"（通过字节跳动·火山方舟接入）**。您也可以接入其他兼容 OpenAI API 协议的引擎。
- **云端身份服务**（仅云端模式）：用于账号注册、登录、组织成员管理。
- **更新服务**：用于检查并下载本软件的版本更新。

4.2 这些第三方服务由第三方独立提供，受其自身服务条款与隐私政策约束。**当您使用涉及第三方服务的功能时，请同时阅读并同意相关第三方的协议**。我们不对第三方服务的可用性、内容、安全性承担责任，但会尽最大努力筛选合规、可靠的合作方。

4.3 第三方服务清单详见《[隐私政策](./02-privacy-policy.md)》"第三方服务"一节。
