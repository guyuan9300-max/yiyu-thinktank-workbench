## 5 · 关键决策点等顾源源拍板

下面 4 件事 v2.2 Phase 2 必须澄清, 不澄清就完不成 N2:

1. **路径 4 (手机 AI 聊天) 怎么 normalize 进 atomic_facts?**
   场景: 用户说"我觉得日慈现金大概剩 300 万"
   → 应该写为 atomic_fact 含 `source_type='user_observation'` + `content_role='observation'` + `verification_status='unverified'`?
   还是写为别的形态 (例如单独存 ai_dialogue_facts 表)?

2. **F2.1 LLM extractor 的 prompt 怎么让 AI 判断 content_role?**
   顾源源给 5-10 个真实案例: 哪些是 fact / decision / risk / observation, AI 才能学准。

3. **朋友式澄清的 prompt 样本?**
   顾源源给 3-5 个真实"朋友式 vs 问卷式"对比, 让 F2.5 能跑通。

4. **AI Memory 写入是 Phase 2 落地, 还是留 3.0?**
   v2.2 NORTH_STAR §3 N3 写"v2.2 阶段只 CREATE + 单向写入, 不读取"。但 F2.1 跑一次 LLM 抽取时, 是否应该顺手往 ai_episode_log 写一条? 还是 v2.2 也不写, 留 3.0?
