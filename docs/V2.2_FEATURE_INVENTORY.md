## 三、跨端核心契约

| 契约 | 校验动作 |
|---|---|
| 桌面改任务 → 手机自动拉到 | 桌面改 task title → 30s 内手机刷新看到 |
| 手机改任务 → 桌面自动同步 | 手机改任务 → 30s 内桌面看到 |
| 多账号隔离 | 账号 A 改的 task 在账号 B 完全看不到 |
| 本地 frozen_at 不被云端覆盖 | 多账号同步时本地 frozen 字段稳定 |
