# 架构概览

益语智库工作台采用桌面端、本地后端和云端后端组合架构。

## 桌面端

- Electron 主进程负责窗口、运行时、打包更新、外链打开和本地服务启动。
- React/Vite 渲染端负责主要交互界面。
- `src/shared/` 存放跨主进程、渲染端和后端契约使用的共享类型与工具。

## 本地后端

`backend/` 是本机 FastAPI 服务，主要负责：

- 本地文件和资料导入。
- 文档解析、知识索引和检索。
- 本机 AI 编排、资料提炼和情报处理。
- 与桌面端 userData 目录中的本地数据库协作。

## 云端后端

`cloud_backend/` 是组织协作 FastAPI 服务，主要负责：

- 登录、组织、成员、部门和权限。
- 协作任务、事件线、周复盘等共享数据。
- 组织级 AI、飞书、对象存储和语音模型配置。
- 与移动端或其他客户端共享协作状态。

## 数据边界

- 本地资料、用户导入文件和本机知识库默认保留在用户本机。
- 组织协作数据通过云端后端同步。
- AI、飞书、对象存储、语音等外部服务必须由使用者自行配置。
