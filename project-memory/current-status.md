# 益语智库平台 — 当前状态

*最后更新: 2026-04-05*

## 今天完成的主要改动

1. **事件线活动分类** — event_line_activities 表新增 is_key 字段，区分关键事件（新建任务、手动备注、附件上传）和系统痕迹（状态变更、字段更新），默认只展示关键事件
2. **任务详情展开** — 事件线报告中，活动卡片下方展示关联任务的标题+说明
3. **附件展示改造** — 折叠时显示文件类型色块+完整文件名；展开时显示文件图标卡片+AI文档摘要（全文，不截断）
4. **图片 OCR 识别** — 展开图片时自动调用 Ark API 做 OCR，结果显示在图片下方
5. **本地附件缓存** — ~/Library/Application Support/YiyuThinkTankWorkbench/Cache/event-line-attachments/，首次从云端拉取后缓存到本地，后续秒加载
6. **Word 导出封面** — 专业封面页（品牌色 #5B7BFE，统计卡片，分类标签），内容页匹配预览状态
7. **事件线说明可编辑** — textarea + 800ms debounce 自动保存到云端
8. **周复盘新壳** — 固定高度布局，tab 切换（重点事件线/AI摘要），视角胶囊（全局/CEO/部门/个人），内容区独立滚动，右下角固定生成按钮
9. **技能库建设** — 安装 engineering 插件 + 创建 5 个自定义技能（react-native-expo, electron-builder, drawio-architect, mermaid-diagram, project-memory）
10. **项目记忆系统** — 建立跨会话上下文持久化
